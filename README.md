# Компонент-календарь (тестовое задание)

Описание
- Тестовое задание: разработать компонент-календарь на Vue.js.
- Функционал: переключение месяцев, выбор даты кликом, возврат выбранной даты через событие, инициализация по входной дате, поддержка смены языка (названия месяцев и дней недели).
- Если входная дата не передана — используется текущая дата.

Установка
1. Клонировать репозиторий:
   - `git clone https://github.com/darkselia/calendar_vue3.git`
2. Установить зависимости:
   - `npm install`
3. Запустить локально:
   - `npm run dev`
